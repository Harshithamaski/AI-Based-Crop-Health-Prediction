# AI-Based Crop Health Prediction
This project detects plant diseases using deep learning (ResNet-50).
## Features
- Upload leaf image
- Predict disease
- Show confidence score
##  Tech Stack
- Python
- Flask
- TensorFlow / Keras
- HTML, CSS, JavaScript
##  Model
- ResNet-50 (Transfer Learning)
- Dataset: PlantVillage
##  How to Run
1. Install requirements
2. Run app.py
3. Open localhost
